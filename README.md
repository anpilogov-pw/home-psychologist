# home-psychologist
Home психолог — это тема для WordPress, разработанная для блога или информационного ресурса, посвящённого психологии, саморазвитию и книгам, которые вдохновляют и меняют взгляд на жизнь. Репозиторий содержит исходный код темы, адаптированной для удобной публикации контента, с акцентом на читабельность, минимализм и пользовательский комфорт.
